# v0.1.0 フィードバック整理

更新日: 2026-03-10

v0.1.0 公開後の要望、修正点、継続課題を 1 か所に集約するためのメモです。
新規に届いた内容と、公開時点から残っている課題を分けて管理します。

## 運用ルール

- 1件ごとに `FB-xxx` または `CF-xxx` の見出しを付ける
- `種別` は `要望` / `バグ` / `UX` / `ドキュメント` / `調査` のいずれかを使う
- `状態` は `未整理` / `未着手` / `確認中` / `対応予定` / `対応中` / `完了` / `見送り` を使う
- 必要に応じて `発生条件` `再現手順` `参考リンク` `関連コミット` を追記する
